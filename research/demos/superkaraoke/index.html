Audio demos - analysis outputs

The following audio examples demonstrate the output of the analysis method in Superkaraoke. The user-selected song is the input audio for the method. After the Superkaraoke analysis, the separated accompaniment contains the audio without the singing. The separated vocals contain the original singing performance in the user-selected audio. In the note-level transcription, the left channel contains the user-selected audio and the right channel contains the transcribed melody synthesized with piano.

A Superkaraoke application

The following video demonstrates one possible application which uses Superkaraoke. The user has selected a song ("Se on tosi jees" by Egotrippi) and used the Superkaraoke analysis to produce the accompaniment and the note-level transription. Then the user sings along with the accompaniment (you may listen to the user singing signal here), and during the performance, the user singing is tuned to the note-level transcription.

Some notes on the application window
Red line: note-level transcription (the fundamental frequency trajectory). This is the target to which the user singing is tuned.
Green line: the fundamental frequency trajectory of the user singing.
